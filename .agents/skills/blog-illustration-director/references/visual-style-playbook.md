# Visual Style Playbook

Use this file only when you need help choosing or sharpening the visual direction.

## Recommended Style Lanes

### 1. Editorial Tension

Best for:
- breaking news
- industry shifts
- controversy
- AI power-balance stories

Characteristics:
- bold single idea
- clear subject hierarchy
- cinematic but restrained
- dark neutrals or steel-blue palette
- negative space for headline overlays

Prompt building blocks:
- editorial magazine cover
- dramatic but realistic lighting
- minimal clutter
- sharp focal point
- high contrast with clean composition

### 2. Productive Tech Realism

Best for:
- tool roundups
- workflow posts
- AI product explainers

Characteristics:
- believable desk scenes
- screens implied, not fabricated
- subtle motion and depth
- softer palette

Prompt building blocks:
- modern workspace
- product-minded composition
- realistic materials
- no fantasy cyberpunk elements

### 3. Diagram First

Best for:
- technical explainers
- architecture posts
- security workflows
- comparisons

Characteristics:
- clean shapes
- labeled flow
- minimal color set
- optimized for comprehension

Prompt building blocks:
- flat editorial infographic
- clean vector layout
- high legibility
- simple hierarchy

### 4. Data Poster

Best for:
- trend analysis
- benchmark comparisons
- market maps

Characteristics:
- poster-like framing
- chart-led composition
- strong typography space
- minimal decorative noise

Prompt building blocks:
- editorial data poster
- elegant information design
- restrained palette
- print-quality composition

## Anti-Patterns

Avoid these unless the user explicitly wants them:

- glowing humanoid robots
- neon brains
- random floating UI panels
- fake screenshots of branded products
- generic code rain backgrounds
- overly literal "AI taking over the world" imagery

## Output Heuristics

- One hero image is usually enough.
- Two strong section visuals beat five weak filler images.
- Use a diagram when the article explains a process, chain of events, or system behavior.
- Use stock-photo searches only as fallback, not as the main creative direction.
- Alt text should describe the meaningful content, not decorative style alone.
