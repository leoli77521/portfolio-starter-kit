# Source Priority

Use this order when collecting AI and technology topics:

## Tier 1: Best sources

- Official company blogs
- Release notes
- Product pages
- Documentation
- Research papers
- Benchmark leaderboards run by credible orgs

## Tier 2: Strong supporting sources

- Major tech publications
- Well-sourced industry newsletters
- Interviews with named executives or researchers

## Tier 3: Signal-only sources

- X posts
- Reddit threads
- YouTube commentary
- LinkedIn posts

Use Tier 3 only to spot a lead, never as the only evidence.

## Query Families

- `[company or tool] release`
- `[tool or model] launch`
- `[tool or model] benchmark`
- `[tool or model] pricing`
- `[tool or model] docs`
- `AI developer tools latest`
- `AI tools this week`
- `site:openai.com OR site:anthropic.com OR site:googleblog.com`

## Rejection Rules

- No rumor-only stories
- No "top 10 tools" listicles unless there is a new event
- No recycled trend pieces without a dateable trigger
